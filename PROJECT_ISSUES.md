## 重要约束

1. **Python 3.12+** 目标版本，公共接口必须保留类型注解
2. **async-first** 架构，所有长时间操作使用 asyncio
3. **测试质量门禁**：新增代码必须有对应的单元测试与集成测试
4. **文档同步**：模块变更必须同步 docs/architecture.md 与 SKILL 文件
